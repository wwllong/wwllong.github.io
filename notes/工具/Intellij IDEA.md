# Intellij IDEA

## 操作快捷键

|快捷键|说明|
|:---|---|
|shift + F6|批量修改|
|ctrl + alt + l|代码格式化|
|ctrl + d|复制上一行|
|ctrl + w|嵌套代码/标签内容全选|
|F2|快速定位到下一个有问题第的代码|
|ctrl + alt + B|查找接口的实现类|
|ctrl + p|函数操作提示|

## 代码生成快捷键

|快捷键|说明|
|:---|---|
|psvm|创建main函数|
|sout|输出语句|
|alt + insert|生成代码快捷键|
|div.container + tab|html生成div+class代码|

## debug快捷键

|快捷键|说明|
|:---|---|
|alt + F8|查询变量|
