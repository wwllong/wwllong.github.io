## 自动内存管理机制

>Java与C++之间有一堵动态分配和垃圾收集技术所围成的高墙，墙外的人想进去，墙里面的人想出来。

Java的自动内存管理机制学习JVM的重要一节。JVM是java程序的内存管理的“皇帝”，了解虚拟机如何管理内存，方便java程序员检查内存泄漏和内存溢出方面的问题。这一部分核心内容为内存区的划分与垃圾收集器技术，也是面试常考的知识点。
