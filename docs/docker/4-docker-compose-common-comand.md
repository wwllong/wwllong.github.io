# Docker Compose 常用命令



## 常用命令一览

| 命令说明       | 命令                 |
| -------------- | -------------------- |
| 前台运行       | docker-compose up    |
| 后台运行       | docker-compose up -d |
| 启动           | docker-compose start |
| 停止           | docker-compose stop  |
| 停止并移除容器 | docker-compose down  |
